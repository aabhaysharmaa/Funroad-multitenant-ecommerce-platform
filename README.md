# Funroad -  Multi-Tenant E-commerce Platform

A robust multi-tenant e-commerce platform designed for vendors and merchants, featuring custom storefronts, advanced product management, and seamless payments. Built with **Next.js 15**, **TailwindCSS v4**, and **ShadcnUI** components, this platform provides a scalable and user-friendly solution for online businesses.

## Key Features

### Multi-Tenant Architecture
- Isolated vendor subdomains
- Custom merchant storefronts

### Payments & Monetization
- Stripe Connect integration
- Automatic platform fee calculation

### Product Management
- Product ratings and reviews
- Category and product filtering
- Search functionality
- Image upload support

### User Experience
- User purchase library for order tracking
- Role-based access control for different user types

### Dashboards
- Admin dashboard for platform management
- Merchant dashboard for storefront management

### Backend
- Powered by Payload CMS for flexible content management

## Tech Stack

- **Frontend:** Next.js 15, TailwindCSS v4, ShadcnUI components
- **Backend:** Payload CMS
- **Payments:** Stripe Connect
- **Architecture:** Multi-tenant, role-based access control
